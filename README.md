# cnweb-wccup2026-journal

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态网页形式存放，内容涵盖不同主题与用途，不针对任何特定域名或网站。

## 项目简介

cnweb-wccup2026-journal 是一个以 HTML 页面归档为核心的仓库。所有页面均为独立构建，可直接通过浏览器打开查看。仓库本身不提供动态服务或后端逻辑，仅作为静态内容存储与分发使用。

## 目录结构

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

目录结构可能随内容更新而调整，请以实际仓库为准。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可离线打开或通过 GitHub Pages 等静态托管服务访问。
- 页面间无强依赖关系，可按需单独使用。
- 部分页面可能引用 `assets/` 目录下的公共资源，建议保持相对路径完整。

## 维护说明

- 本仓库不定期更新，新增或修改页面时会在提交记录中注明。
- 欢迎提交 Issue 或 Pull Request 以修正错误、改进内容或添加新页面。
- 请勿在仓库中上传含恶意代码、广告或侵犯他人权益的内容。
